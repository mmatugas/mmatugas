Mini Goals Web Application




Hello, readers! For this project I decided to do a to-do list with a clock on the side and an additional motivational quotes that could boost your productivity. I learned to create a to-do list from Code Explained from youtube and the clock from Online Tutorials also from Youtube. I divided my code in to three parts the Model, View, and Controller. I was a little bit confused at first but I tried to understand it as much as I can. All in all, I had fun doing this project.

Functionality




  1. Add a task or goal.
    • Type in your task or goal in the input box then press Enter.
  2. Strikethrough a task or goal.
    • Click the circle beside a task and it will automatically strikethrough
      the task and it will change the button to check button.
  3. Delete a task.
    • Click the x button at the end of a task and it will automatically delete
      the task.
  4. Clear everything.
    • Click the refresh button at the upper right corner of the header
      and it will automatically clear up everything.

    5. TAKE NOTE: Quotation will automatically change every task input.
