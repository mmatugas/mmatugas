Trivia Quiz




Hello, readers! For this project, I did a Quiz Trivia App built with REST API. To be honest, this is the hardest project for me so far. I thought a lot of application I could do however everything I have thought were all beyond of what I can do. So I decided to stick with the class lab as a reference. As always, I tried to understand it as much as I could and kind of change it a little bit.

Functionality




  1. Play
    • Play button lets the player start the game.
  2. Questions
    • Click the correct answer (text) and if it is correct
      page will load to the next question.
    • Topics are randomized, from entertainment to science and many more.
  3. Points
    • Points will be added based of the time the player answered the question.
  4. Time
    • 10 seconds will be added when questions are answered correctly.
    • Once the time reach 0 second that means the game is over
  5. Skip
    • Skip button lets the player skip a question
  6. End
    • Highscores are shown at the start menu
    • Top 5 score will only be kept.
    • Lets the player submit its name when the player scored
      higher than the top 5.


Future Improvement:

  1. I would like to add a difficulty level and topic choices for the player
    so the game has more options in to it.
